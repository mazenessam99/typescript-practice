## ⚡ Smart Run System (Recommended)

Instead of running long paths or navigating folders manually, this project uses a smart runner system.

### 🚀 Run Any Problem

npm run p <folder> <problem-number>

### 📁 Examples

Intro Types:
npm run p intro-types 01  
npm run p intro-types 02  

Data Types:
npm run p data-types 03  
npm run p data-types 04  

Interfaces & Classes:
npm run p interfaces-class-1 01  
npm run p interfaces-class-1 02  

Generics:
npm run p class-generics 01  
npm run p class-generics 02  

---

## 🧠 How It Works

- Uses custom Node.js runner (run.js)  
- Automatically builds path:
  src/<folder>/problem-<number>/solution.ts  
- Executes using ts-node  
- No need for dist or manual navigation
